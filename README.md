# 🧸 Flipzy Toy Store – MERN Stack eCommerce Website

Flipzy Toy Store is a full-featured eCommerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

The platform allows users to browse toys, register/login securely, add products to cart, and place orders.

It also includes an Admin Panel where the admin can manage products, users, and orders efficiently.

---

## 🚀 Features

* 🔐 User Authentication (JWT Based Login/Register)
* 🛒 Add to Cart & Checkout System
* 📦 Order Management
* 👤 User Dashboard
* 🛠 Admin Panel (Manage Products & Orders)
* 📱 Responsive UI Design
* 🌐 REST API Integration

---

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Node.js + Express.js
* Database: MongoDB
* Authentication: JWT
* Styling: CSS / Bootstrap / Tailwind (if used)

---

## 🎯 Project Purpose

This project was built to demonstrate full-stack development skills using the MERN Stack and to understand real-world eCommerce application architecture.
